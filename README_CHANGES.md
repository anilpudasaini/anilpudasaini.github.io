# Anil Pudasaini Portfolio — Interactive Revision v2

This revision updates the static portfolio into a more polished, interactive website while keeping it compatible with GitHub Pages and normal static hosting.

## Main changes

- Replaced the hero image with the provided professional portrait: `img/anil-hero.png`.
- Removed the voice privacy slider and all draft/template-style helper text.
- Added a premium hero section with animated background, cursor glow, rotating role title, interactive focus buttons, professional portrait treatment, and role-specific visual badges.
- Kept the light/dark mode toggle.
- Moved filtering to the Experience section with filters for Systems Support, Teaching, Research, and Development.
- Revised the skills section using the uploaded Teaching CV and IT Support CV.
- Updated selected work to cover systems support, automation, AI research, machine learning, and software projects.
- Added linked publication cards using DOI/publisher/conference links where available.
- Kept the project as a static HTML/CSS/JS site.

## Key files changed

- `index.html`
- `css/portfolio.css`
- `js/portfolio.js`

Upload the contents of this folder to your GitHub Pages repository or static hosting account. The file `index.html` remains the main entry point.
